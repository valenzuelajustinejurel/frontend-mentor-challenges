## My approach

- I build every challenge independently — no AI generating code for me
- I use [javascript.info](https://javascript.info) and [web.dev](https://web.dev/learn) as references when I get stuck on a concept
- After finishing, I review other people's solutions to learn different approaches
- Each challenge is deployed so you can see it live, not just read the code
